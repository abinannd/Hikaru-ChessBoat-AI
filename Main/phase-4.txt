Phase 4 – Step 8: Human vs AI Command-Line Gameplay

Objective:
Create a standalone CLI application that allows a human to play chess against the trained AI using the existing inference engine.

IMPORTANT:
Implement ONLY Step 8.

Do NOT modify the neural network.
Do NOT retrain.
Do NOT change board encoding or move encoding logic.

----------------------------------------------------
Files
----------------------------------------------------

Create:

Main/src/play_cli.py

Reuse:

- ChessInference
- MovePrediction
- python-chess

----------------------------------------------------
Game Flow
----------------------------------------------------

1. Start from the standard chess initial position.

2. Display the board using python-chess.

3. Ask the user whether they want to play:

White
or
Black

4. Initialize a chess.Board().

----------------------------------------------------
Human Turn
----------------------------------------------------

Prompt:

Enter your move (UCI):

Examples:

e2e4
g1f3
e7e8q

Validate:

- UCI syntax
- Move legality

If invalid:

Explain the error.

Ask again.

Do NOT crash.

----------------------------------------------------
AI Turn
----------------------------------------------------

Call:

predict_best_move(board.fen())

Receive a MovePrediction.

If move is None:

End the game.

Otherwise:

Display:

AI chooses: <move>

Push the move onto the board.

----------------------------------------------------
Board Display
----------------------------------------------------

After every move print:

Current board

FEN

Side to move

Move number

Game status

----------------------------------------------------
Game Over
----------------------------------------------------

Detect:

Checkmate

Stalemate

Insufficient material

Fifty-move rule

Threefold repetition

Print the correct result and terminate cleanly.

----------------------------------------------------
Move History
----------------------------------------------------

Maintain a move history.

At game end print:

1. e2e4
1... e7e5
2. g1f3
...

----------------------------------------------------
Testing
----------------------------------------------------

Verify:

Human as White

Human as Black

Invalid move input

Illegal move input

Checkmate

Stalemate

Game termination

----------------------------------------------------
Constraints
----------------------------------------------------

Reuse the existing inference engine.

Do NOT duplicate prediction logic.

Do NOT modify board state outside normal chess moves.

Keep play_cli.py independent of the AI internals.

Future GUI or web interfaces should be able to reuse the same ChessInference API.
