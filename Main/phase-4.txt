Phase 4 – Step 6: Legal Move Filtering

Objective:
Convert the neural network's raw predictions into a single legal chess move by evaluating every legal move using the network's output scores.

IMPORTANT:
Implement ONLY Step 6.

Do NOT implement gameplay, CLI, evaluation, Stockfish integration, or GUI.

Reuse all existing modules.
Do NOT rewrite board_encoder.py or move_encoder.py.

----------------------------------------------------
Requirements
----------------------------------------------------

1. Modify:

Main/src/inference.py

2. Add a method:

get_best_legal_move()

3. Preconditions:

- A board has already been loaded.
- The board has already been encoded.
- The network has already produced logits.
- The move decoder already exists.

If any prerequisite is missing, return a clear error.

----------------------------------------------------
Legal Move Generation
----------------------------------------------------

Generate all legal moves using:

board.legal_moves

Store them in a list.

Print:

Number of legal moves found.

----------------------------------------------------
Move Scoring
----------------------------------------------------

For EVERY legal move:

1. Encode the move using the existing move_encoder.py.

2. Obtain its class index.

3. Read the corresponding logit directly from the model output.

Example:

score = logits[class_index]

Do NOT run the network again.

Use the already computed logits.

----------------------------------------------------
Move Selection
----------------------------------------------------

Compare the scores of every legal move.

Select ONLY the legal move with the highest logit.

Return:

- chess.Move
- UCI string
- class index
- raw logit score

----------------------------------------------------
Verification Output
----------------------------------------------------

Print:

Logit score

Top 10 legal moves ranked by score
